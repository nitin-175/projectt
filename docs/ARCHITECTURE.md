# Architecture Notes

## Frontend

- React + Vite
- Tailwind CSS
- React Router
- Zustand for session and UI state
- Axios for API communication

## Backend

- Spring Boot
- Spring Security
- JWT authentication
- Spring Data JPA
- MySQL
- WebSocket/STOMP for real-time seat updates

## Shared Packages

- `@show-booking/types`: request/response shapes and enums
- `@show-booking/ui`: reusable presentational components
- `@show-booking/utils`: formatting and helper functions
